# UrbanRentals – Online Furniture and Home Appliances Rental System

Overview

UrbanRentals is a web-based rental management system developed using Django. It enables users to rent furniture and home appliances for a specified period instead of purchasing them, providing an affordable and convenient solution for students, professionals, and families.

The system allows customers to browse rental products, select rental durations, place bookings, and make payments through an intuitive interface, while administrators can efficiently manage products, categories, customers, rentals, and payments.

# Features

**Customer**

User registration and login

Browse furniture and home appliances

View product details

Select rental duration

Place rental bookings

Make payments

View rental history

Manage profile

 **Administrator**

Manage categories

Manage rental products

Manage customers

View rental bookings

Manage payments

# Running the Project

Clone the repository

git clone https://github.com/<your-username>/UrbanRentals.git
cd UrbanRentals

**Create a virtual environment**

python -m venv venv

**Activate the virtual environment**

Windows

venv\Scripts\activate

**Install dependencies**

pip install -r requirements.txt

**Create a MySQL database and update the database credentials in settings.py.**

**Apply migrations**

python manage.py migrate

**Run the development server**

python manage.py runserver

**Open your browser:**

http://127.0.0.1:8000/
